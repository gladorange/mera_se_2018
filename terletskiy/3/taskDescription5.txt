Пятое задание:
Будем создавать предприятие по управлению различными машинами.
- Создать интерфейс  Vehicle. В нем объявить метод moveTo(String newLocation). Текущее местоположение должно быть запомнено.
- Создать интерфейс CargoVehicle, который расширяетт интерфейс Vehicle методом loadCargo(String cargo) и методом getMaxWeight().
- Создать абстрактный класс Car, реализующий интерфейс Vehicle поля "расход топлива на километр",
- Создать абстрактный класс CargoCar - поле максимальная грузоподъемность. Родитель это класса - Car
- Создать абстрактный класс Ship, реализующий интерфейс Vehicle и  имеющий поле осадка (Draft )
-Создать абстрактный класс CargoShip имеющий поля максимальная грузоподъемность, наследник Ship
- Cоздать класс Motorbike (Vehicle), Truck (Cargo Vehicle),  Yacht (Ship), Barge (CargoShip).

Метод loadCargo должен запоминать название груза, переданного в качестве аргумента.
Метод moveTo должен выводить что-то вроде "Еду туда-то из такого-то города" или "Плыву туда-то из такого-то города" в зависимости от типа транспортного средстава.
Причем, если у ТС есть груз, то по прибытии в пункт назначения надо вывести "Доставил <название груза> в <место>". Груз после этого должен обнулится, местоположение тоже должно изменится.

Создать парк транспортных средств (в массиве) - как инимум один экземпляр каждого транспортного средства.

Направить все ТС в какой-то город.
Направить все грузовые ТС в другой город без груза.
Загрузить грузовые ТС чем-нибудь
Вернуть их обратно в город.
